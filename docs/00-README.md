# PathWise Project Overview

- **What:** A two-sided tutor-student marketplace with personalised roadmaps, tutor matching, course creation, and session booking.
- **State:** ~85% MVP complete. Core flows work (quiz → roadmap → matching → booking). Missing: real-time messaging, payment integration, admin analytics.
- **Domain:** pathwise.network (Vercel deployment)
